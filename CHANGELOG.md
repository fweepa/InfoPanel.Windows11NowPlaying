# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-11

### Added

- Initial release
- Display currently playing track title, artist, and album from Windows 11 GMTC
- Show source application name
- Playback position, duration, and percentage (0–100) for gauge display
- Playback status (Playing, Paused, Stopped, etc.)
- Smooth position extrapolation when playing between API polls
- Volume sensor (placeholder; GMTC does not expose volume)
